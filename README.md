# ⚽ PITCH BOSS (v1.1)

> Own the Pitch. Run the Club.

Browser-based football management game with a heavy Neon-Punk aesthetic.
Built with Vanilla JS, HTML5 Canvas and Web Audio API.
Zero dependencies. No build step.

## Play
🎮 [Play on Itch.io](https://nando-mbp13.itch.io/pitch-boss)
🌐 [Play on GitHub Pages](https://nando-mbp13.github.io/pitch-boss/)

## About
Take control of a football club from scratch.
Sign players, set tactics, simulate matches with a cinematic engine,
and climb the league table in 9 brutal jornadas to lift the trophy.

## Features (v1.1 Update)
- **Cinematic Match Engine:** Real-time ball tracking, positional player dots, and dramatic slow-motion goals on an HTML5 canvas.
- **Dynamic Squad Management:** Manage an 18-man procedural roster and balance your wage bill.
- **Dual Aesthetic:** Custom high-contrast Dark UI and a toggleable "Cyber-Day" light theme.
- **Procedural Audio:** 100% of sound effects generated live via Web Audio API.

## Built with
- Vanilla JavaScript — zero frameworks
- HTML5 Canvas API — match animation
- Web Audio API — procedural punk sound
- CSS3 — Neon-Punk street UI

## Run locally
Open `index.html` in any browser. That's it.
